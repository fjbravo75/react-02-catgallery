# CatGallery - Proyecto React

Proyecto del módulo ReactJS basado en la entrega `CatGallery` realizada antes con JavaScript.

La aplicación permite ver imágenes reales desde The Cat API, filtrar por tipo o raza, guardar favoritos en `localStorage`, ver una página de favoritos y abrir una ficha de detalle.

## Estructura

- `index.html`: página principal.
- `favoritos.html`: página de favoritos.
- `src`: código fuente React.
- `src/components`: componentes reutilizados.
- `src/pages`: vistas principales.
- `src/styles.css`: estilos de la aplicación.
- `docs`: versión generada para GitHub Pages.

## Tecnología usada

- Vite.
- React.
- JavaScript/JSX.
- CSS propio.
- The Cat API.
- `localStorage`.

## Comandos

```bash
npm install
npm run dev
npm run build
```
